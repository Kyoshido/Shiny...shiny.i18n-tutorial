What are ClinRTools?

ClinRTools help clinical trial specialists to plan new studies.

Demo version of ClinRTools is distributed without any warranty and should be used at your own risk. Please remember to verify/validate the results with the biostatisician.

Technical details

ClinRTools use R language to provide all the computations.

R sessions are running on the server, so users are not required to have a local installation.

Reproducible code and environment options can be found in each of the Tools pages.

Help and Feedback
