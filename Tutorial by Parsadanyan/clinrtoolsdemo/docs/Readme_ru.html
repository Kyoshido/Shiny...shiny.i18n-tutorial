Что такое ClinRTools?

ClinRTools - это набор приложений для планирования клинических исследований.

Демонстрационная версия содержит ограниченное число приложений. Данный демонстрационный набор распространяется без какой-либо гарантии и должен использоваться на свой страх и риск.

Пожалуйста, обращайтесь к ответственному биостатистику для проверки и валидации расчетов.

Техническая информация

Все расчеты в ClinRTools производятся с использованием статистического пакета R.

Запуск приложения происходит на сервере, поэтому пользователям не нужно устанавливать R на рабочий компьютер.

Код для воспроизведения расчетов, а также системную версию R можно получить непосредственно в приложении.
